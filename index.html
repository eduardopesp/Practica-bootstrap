Tipos

Cada tipo de bici acorde a tus necesidades

Mountain bike

Es tu bici perfecta para excursiones por la montaña o el campo. Tiene ruedas resistentes para cualquier tipo de superficie

Paseo

Si lo que buscas es una bici para ir por la ciudad esta es la ideal. Te permite una posicion mas comoda con la espalda mas recta, ideal para trayectos cortos.

Carretera

Esta diseñada para alcanzar altas velocidades, tiene una rueda estrecha para que agarre menos en el asfalto. Sólamente para terrenos lisos
